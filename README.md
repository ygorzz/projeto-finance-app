# 💰 FinanceApp - Gestor de Finanças Pessoais

O **FinanceApp** é uma aplicação web completa para controle financeiro, desenvolvida como projeto de conclusão do curso **CS50: Introduction to Computer Science** da Universidade de Harvard. O objetivo do projeto é oferecer uma interface intuitiva para que usuários possam gerir seus orçamentos, categorias e fluxos de caixa de forma organizada.

## 🚀 Funcionalidades

* **Autenticação:** Sistema de login e registro de novos usuários.
* **Gestão de Categorias:** Criação e personalização de categorias para gastos e ganhos.
* **Orçamentos:** Definição de limites financeiros por categoria.
* **Fluxo de Caixa:** Lançamento de entradas e saídas de valores.
* **Histórico de Transações:** Visualização detalhada de todas as movimentações realizadas.
* **Dashboard Dinâmico:** Resumo visual da saúde financeira utilizando gráficos interativos.

## 🛠️ Tecnologias Utilizadas

* **Front-end:** [Bootstrap 5](https://getbootstrap.com/), HTML5, CSS3.
* **Back-end:** [Python](https://www.python.org/) com o micro-framework [Flask](https://flask.palletsprojects.com/).
* **Banco de Dados:** [SQLite](https://www.sqlite.org/) para persistência de dados.
* **Interatividade:** JavaScript (ES6+) para validações dinâmicas de formulários, eventos e requisições assíncronas (Fetch API).
* **Gráficos:** [Chart.js](https://www.chartjs.org/) para o dashboard dinâmico.
